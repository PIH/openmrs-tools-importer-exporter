
# PIH OpenMRS Importer - Exporter App

## Quick start

### Prerequisites

- Node v.20

Create a ".env" file from ".sample-env" and include the proper credentials, etc

### Install and Run

`npm install`

`node exportUsers.js`
`node importUsers.js`

`node exportProviders.js`
`node importProviders.js`

`node exportPatients.js
`node importPatients.js
`
